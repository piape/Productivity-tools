-------------------------------------------------------------------------------
Neurosound Tracker_v1.0
-------------------------------------------------------------------------------
# Before use
    1. You should install python => 3.0 above.

-------------------------------------------------------------------------------
# How to use.
    1. python Neurosound_Tracker_v1.0
    2. Choose Video.
    3. Drag and choose pixels.
    4. The function of Space"
	- Play and pause.
	- if you want to reset pause and drag again.
	- if you want to stop, please push "esc" twice.
    5. "output.txt" include range of box moving based on pixels.

-------------------------------------------------------------------------------
# Warnings
    1. Not include Except processing.
    2. Consider axis x only.
    3. You can choose tracking algorithm 0 to 6. (Do not input 7 or above)
