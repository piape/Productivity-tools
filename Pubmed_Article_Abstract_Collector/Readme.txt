-------------------------------------------------------------------------------
Pubmed Abstract Collector
-------------------------------------------------------------------------------
# Before use
    1. You should install python => 3.0 above.

-------------------------------------------------------------------------------
# How to use.
    1. CMD
    2. "python Pubmed_Global.py"
    3. Keyword input // ex. low back pain
    4. from ~ // ex. 2000
    5. to ~ // ex. 2022
    6. Data will save "/collect_list" path.

-------------------------------------------------------------------------------
# Warnings
    1. Not include Except processing.
    2. Depend on Internet speed.
