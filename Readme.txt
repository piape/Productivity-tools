------------------------------------------------------------------------------------------------------------------------------------------------------------
국내 논문 수집기(Riss)
------------------------------------------------------------------------------------------------------------------------------------------------------------
# 사전 설치 필요
    1. python 설치 (v 3.0 이상)

------------------------------------------------------------------------------------------------------------------------------------------------------------
# 설치 후 사용법
    1. 압축을 푼 폴더에서 마우스 오른쪽 > 터미널 실행
    2. python Riss_Korea.py 입력 및 엔터
    3. 원하는 논문 제목 입력 ex. low back pain
    4. 시작 년도 입력 (낮은) ex. 2000
    5. 종료 연도 입력 (높은) ex. 2022
    6. 수집 데이터는 /collect_list 폴더에 저장 됩니다.
    7. 중간에 뜨는 이벤트 로그는 무시하시면 됩니다.
------------------------------------------------------------------------------------------------------------------------------------------------------------
# 주의 사항
    1. 사용자 입력창에 예외처리를 넣지않았음.
    2. 인터넷 속도에 따라 오류가 발생할 수 있음.
    3. 공유 금지.
